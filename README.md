# 🔍 NodeJS Query

Repositório de estudos criado para praticar o uso de **Query Parameters** e manipulação de dados em requisições utilizando **Node.js puro**.

## 🚀 Sobre

Os exercícios presentes neste repositório abordam:

- Query Parameters;
- Request Body;
- Diferença entre rota e query;
- Manipulação de parâmetros em requisições.

## 🛠️ Tecnologias

- Node.js
- JavaScript

## 🎯 Objetivo

Praticar conceitos fundamentais relacionados à passagem e tratamento de dados em requisições HTTP utilizando Node.js.
